About

Want to know about the app ?

It is a simple weather app created in django using python requests. Just enter the city name and get detailed info of its weather. The project is open source and pull requests are welcome.View Source

The app features minimal and flat design. All it takes is just a minute to get the weather info.

Also you can get 7-day weather forecast just a click away.

It includes full HTML, CSS, and custom JavaScript files for easy customization and Python's Django Framework as its backend for fast and secure experience.
